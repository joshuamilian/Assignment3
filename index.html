Benefits of HashMaps

  • With a map, it lets you store a value based on a key that can be used to retrieve an item.

  • HashMap is fail-fast iterator. Where the Iterator is an object that enables a programmer to traverse a container, particularly lists.

  • HashMap cannot be shared between multiple threads without proper synchronization.
